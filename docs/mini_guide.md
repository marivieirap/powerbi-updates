# Mini Guia de Estudos — Power BI Updates

Este guia apresenta como utilizar o notebook construído no NotebookLM para estudar atualizações do Power BI de forma estruturada.

O objetivo é transformar informações técnicas em conhecimento organizado, facilitando o aprendizado contínuo sobre a evolução da ferramenta.

---

# 🎯 Objetivo do Guia

Este guia tem como finalidade:

- organizar conteúdos sobre Power BI
- facilitar o acompanhamento de atualizações da ferramenta
- transformar conteúdos técnicos em materiais de estudo
- apoiar profissionais de dados no aprendizado contínuo

---

# 📊 Resultados Gerados pelo Notebook

O notebook foi estruturado para gerar três tipos principais de materiais de estudo:

## 1️⃣ Resumos Estruturados

O NotebookLM gera resumos organizados sobre atualizações e conteúdos relevantes do Power BI.

Esses resumos incluem:

- descrição das novidades
- explicação do funcionamento dos recursos
- exemplos de aplicação prática
- impacto para profissionais de BI

Exemplo de uso:

Prompt:

Quais são as principais atualizações recentes do Power BI Desktop?

Resultado esperado:

- resumo das novidades
- explicação das funcionalidades
- possíveis aplicações em projetos de dados

---

## 📖 2️⃣ Glossário de Conceitos

O notebook também pode ser utilizado para gerar um glossário com conceitos importantes do ecossistema do Power BI.

Exemplos de conceitos incluídos:

- DAX
- Power Query
- Modelo Semântico
- Dataset
- Dataflow
- DirectQuery
- Refresh de Dados

Prompt exemplo:

Crie um glossário com os principais conceitos utilizados no Power BI.

Resultado esperado:

- definição clara de cada conceito
- explicação simplificada
- exemplos de uso no contexto de BI

---

## 🧠 3️⃣ Conjunto de Prompts

O projeto também inclui um conjunto de prompts utilizados para explorar as fontes e gerar análises relevantes.

Exemplos de prompts:

### Atualizações recentes

Quais são as atualizações mais recentes do Power BI Desktop?

### Impacto prático

Quais atualizações recentes do Power BI podem impactar performance de modelos?

### Explicação de recurso

Explique uma nova funcionalidade do Power BI e como ela pode ser aplicada em projetos reais de BI.

### Resumo mensal

Crie um resumo das principais atualizações do Power BI no último mês.
